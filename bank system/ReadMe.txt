一. 项目依赖配置
运行依赖于 mysql 数据库 数据表 数据
1. 可在 mysql 中运行 java后端源码 -> bank -> mysql.sql 中的 sql 语句，进行建库，建表，初始化一些数据
2. 且需要在 java后端源码 -> bank -> src -> db -> DB 类 -> getConn函数中，修改 mysql 数据库的配置信息

二. 项目运行方法
项目运行在 tomcat 中
运行方式:
1. 执行 tomcat项目运行环境 -> tomcat-8 -> bin -> startup.bat 文件
2. 在浏览器中 输入默认 url: http://localhost:8080/ 即可显示

三. 项目业务简略说明
此项目为 银行自动化交易系统
可在交易页面点击 start 按钮进行自动化交易，每次交易十个回合，每个回合的交易类型，交易用户和交易金额均是随机产生
十回合之后 点击 phrase 按钮，银行会给每个用户计算一次利息，然后跳转到用户信息页面
用户信息页面用来查看用户的账户信息
